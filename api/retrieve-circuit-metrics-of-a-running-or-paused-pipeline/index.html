Path Parameters
pipeline_name string — REQUIRED

Unique pipeline name

Query Parameters
format string

Possible values: [prometheus, json]

Circuit metrics output format.

  • prometheus: format expected by Prometheus
  • json: JSON format

200

Pipeline circuit metrics retrieved successfully
